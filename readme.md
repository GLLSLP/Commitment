영철

```markdown
# 어제 했던 일
    - 커밋지도 개발위한 작업 => 구글지도 불러오기
    - 시행착오 좀 겪음 => 지도스타일을 건드려야하는데 구글지도가 우리나라를 건드릴 수 없게함
# 오늘 할 일
    - 지도 타일을 변경하는 방법으로 해볼 예정
    - 실제 경위도와 타일 좌표가 다른데, 서로간에 변환해주는 공식을 알아보고 함수화할 예정
    - 커밋지도 틀 만들기 목표
# 고충
    - 구글지도를 뷰에서 보이게 하는 것 자체에서 애를 먹음
```

유정

```markdown
# 어제했던 일
db 설계
- erd 작성
- 테이블 정의서 작성
- 테이블 생성
역할분담해서 jira 수정
패키지명과 클래스명 정하고 환경셋팅함
userdto 작성 lombok 기능 사용
현재는회원가입 기능구현중

# 오늘 할 일
회원가입 수정 탈퇴 토큰 사용한 로그인

# 고충
db설계하면서 좀 헷갈리는게 있었다.

```

청화

```markdown
# 어제했던 일

# 오늘 할 일
commit 작성 postman으로 테스트
commit 불러오기
위치기반 SNS

# 고충
JPA에 대해 잘 알지 못해서 1:N, N:1관계를 어떻게 어노테이션으로 해결해야 할 지 먼저 공부해야 할 것 같다.
커밋, 위치기반 SNS 부분을 하고 있는데, user에 대한 코드가 짜여 있지 않은 상태로 하다 보니 임의로 user 관련 코드까지 짜느라 시간이 배로 걸렸다.

```

소현

```markdown
# 어제했던 일

```

기기별로 와이어프레임 조정
